# 使用者操作手冊 (User Guide)

| 版本 | 日期 | 作者 | 說明 |
|------|------|------|------|
| 1.0.0 | 2026-05-20 | Mavis | 初始版本 |

## 系統登入
開啟瀏覽器進入系統首頁，預設提供員工查詢與打卡入口。若啟用權限模組，請輸入帳號密碼登入。

## 一、員工管理 (Employee Management)
### 新增員工
1. 進入 [員工管理] 頁面
2. 點擊「新增員工」
3. 填寫基本資料：姓名、員工編號、部門、職位、Email
4. 系統自動驗證 Email 唯一性，確認後儲存

### 查看與篩選
- 支援依部門、狀態 (在職/休假/離職) 篩選
- 搜尋欄位可輸入姓名或員工編號快速定位
- 點擊員工名稱可查看详细出勤與請假紀錄

### 編輯與離職
- 修改職務或部門資訊需填寫變更原因
- 離職操作會將員工狀態改為 `resigned`，歷史出勤資料保留

## 二、出勤管理 (Attendance)
### 打卡流程
- **上班打卡**：建議於 09:30 前完成，系統自動判定是否遲到
- **下班打卡**：建議於 17:30 後完成，晚於標準工時將視為早退或正常
- 打卡紀錄即時同步至個人出勤月報表

### 異常處理
- 若忘打卡，可透過「補卡申請」提交補正
- 主管審核通過後，系統自動更新當日狀態

### 報表檢視
- 進入 [出勤統計] 可查看月度出勤熱力圖
- 支援匯出 CSV 格式供人事薪資計算使用

## 三、請假申請 (Leave Request)
### 申請步驟
1. 點擊「新增請假」
2. 選擇假別：
   - 年假 (Annual Leave)
   - 事假 (Personal Leave)
   - 特休 (Special Leave)
   - 病假 (Sick Leave)
3. 設定請假日期區間與事由
4. 系統自動計算可用餘額，提交後呈報主管審核

### 審核流程
- 主管收到通知後至 [請假管理] 頁面審核
- 狀態變更為「核准」或「駁回」，員工端即時收到通知
- 核准後自動扣減可用假別天數

## 四、加班申請 (Overtime)
### 時數計算規則
- 系統自動計算 `end_time - start_time` 時數
- 超過 4 小時需填寫詳細事由
- 加班申請需經主管核可後生效

### 查詢與匯總
- 進入 [加班紀錄] 查看個人與部門統計
- 支援依月份、員工篩選
- 匯出功能提供薪資部門核算用之標準格式

## 五、公司行事曆 (Calendar)
### 功能介紹
- 整合國定假日、公司行事曆與會議活動
- 自動標記特殊出勤日 (如補班日)

### 新增事件
- 點擊「新增行事曆事件」
- 選擇類型 (會議/假日/活動)，填寫日期、標題、地點
- 可指定參與人員，系統自動檢查衝突

### 匯入功能
- 支援上傳 ICS/CSV 檔案匯入外部行事曆
- 匯入資料標記為 `is_imported = true`，便於後續管理

## 六、個人中心
- 查看個人出勤統計、請假餘額、加班總時數
- 修改個人基本資料與聯絡方式
- 切換暗色/亮色主題

## 快速鍵與捷徑
| 捷徑 | 功能 |
|------|------|
| `Ctrl+K` | 快速搜尋員工 |
| `Ctrl+L` | 新增請假 |
| `Ctrl+O` | 新增加班 |
| `F5` | 重新整理資料 |
